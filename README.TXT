# SIMRS Khanza

Software ini adalah **Software Manajemen Rumah Sakit/Klinik/Puskesmas** yang **gratis**, sumber tersedia, dan boleh digunakan siapa saja tanpa dikenai biaya apapun.  

⚠️ **Hal yang tidak boleh dilakukan:** memperjualbelikan atau mengambil keuntungan dari software ini dalam bentuk apapun tanpa seizin pembuat software (**Khanza.Soft Media**).
